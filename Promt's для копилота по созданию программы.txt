моё приложение выполняется в WPF с помощью ADONet и EntityFramework. Размести все элементы в WPF через GRID. Сделай 5 столбцов и необходимое количество строк. Сделай выпадающий список для каждого из элементов Стены, Колонны, Перекрытия, Переходные через Combobox и TextBox's для каждого из параметров этих элементов, которые будут заполняться данными в зависимости от выбранного элемента в ComboBox. Каждый элемент Combobox и Textbox должен быть подписан Label. 4 столбца используются  для каждого из элементов Стены, Колонны, Перекрытия, Переходные, пятый столбец используется для вывода Изображения, хранящегося в базе данных в зависимости от выбранных элементов. То есть максимально может быть выведено до 4 изображений одновременно, минимально 0, если не выбран ни один из элементов.
Для каждого из элементов в каждом столбце необходимо сделать четыре кнопки: Изменить, Создать, Удалить, Очистить поля ввода, которые соответственно для каждого из элементов будут выполнять следующую логику: Изменять данные выбранного элемента в базе данных, Создавать новый элемент в базе данных, Удалять выбранный элемент из базы данных, Очищать поля ввода данных без удаления из базы данных. При нажатии на кнопку Удалить, должно появляться Окно предупреждения в качестве защиты от случайного нажатия с кнопками Удалить и Отмена. В приложении в самом низу должно быть поле для ввода SQL запроса к базе данных и окно для вывода результатов запросов. Также должен быть выпадающий список в котором будут прописаны фиксированные запросы к базе данных и отображаться они должны в текстовом виде (сам текст запроса в выпадающем меню не должен быть виден пользователю): Суммарное количество бетона всех конструкций на секции №4, Суммарное количество бетона всех конструкций на секции №5, Суммарное количество бетона класса В25 всех конструкций на секции №4, Суммарное количество бетона класса В25 всех конструкций на секции №5, Количество бетона всех конструкций 5го этажа на секции №4, Количество бетона всех конструкций 5го этажа на секции №5, Неповторяющееся количество классов бетона на секции №4, Неповторяющееся количество классов бетона на секции №5. Результаты запросов этого выпадаюзего списка должны также выводиться в окно для вывода результатов запроса.
